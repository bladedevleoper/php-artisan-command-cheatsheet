PHP Artisan Commands:

PHP Artisan Help

This will show the all available commands that we can use within the command line:

Command: php artisan --help
Make a Controller

Make a model and create a controller and a migration for that model.

Command: php artisan make:model SingularName -c -m
Make a controller:
command line: php artisan make:controller SingularNameController
Make a model:

(don't forget to think 'singular' when naming models)

Command: php artisan make:model SingularName
Make a migration

(dont forget to make this a plural which means it has an s on the end of the table name), for example create_customers_migration

Command: php artisan make:migration create_nameoftable_migartion
Run Migrations

To run migrations that have not been run then run this command. This won't drop any tables, just run anything that has not been migrated.

Command: php aritsan migrate
Run Migrations when alerations are made to a migration

If alterations are made to a migration that has already been run then we can simply rollback our migrations and migrate again.

Command: php artisan migrate:refresh
